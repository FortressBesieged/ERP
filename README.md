# 技术框架
* 核心框架：SpringBoot 2.0.0
* 持久层框架：Mybatis 1.3.2
* 日志管理：Log4j 2.10.0
* JS框架：Jquery 1.8.0
* UI框架: EasyUI 1.3.5
* 桌面框架: HoorayOS 2.0.0
* 项目管理框架: Maven 3.2.3

# 开发环境
建议开发者使用以下环境，可以避免版本带来的问题
* IDE: IntelliJ IDEA 2017+
* DB: Mysql5.7.4
* JDK: JDK1.8
* Maven: Maven3.2.3+
